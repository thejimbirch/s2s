# Tables Documentation

More information about bootstrap 5 tables can be found here https://getbootstrap.com/docs/5.0/content/tables/

## Updating Tables Variables
Table variables can be changed in **_tables.scss** found in **components/bootstrap/base/tables/_tables.scss**.

The available variables can be found in bootstrap 5 Documentation:
- https://getbootstrap.com/docs/5.0/content/tables/#sass

### Example changing the table font color:

    $table-color: purple;
